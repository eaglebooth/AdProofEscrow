# Aurivus-Inspired Design Guidelines

## Visual Direction

AdProof Escrow uses a dark emerald revenue-ops aesthetic inspired by the provided Aurivus screenshots:

- Deep green-black canvas with subtle grid lines.
- Bright emerald accents for status, CTAs, icons, and active timeline nodes.
- Large centered hero typography with a single highlighted keyword.
- Thin bordered cards with low-contrast green panels.
- Dotted construction lines and crosshair corners around sections.
- Rounded media panel with a creator-campaign preview and play/control affordance.

## Palette

- Background: `#03140f`, `#061d16`
- Panel: `#08251e`, `#0b2c24`
- Line: `rgba(54, 130, 106, 0.28)`
- Dotted line: `rgba(58, 153, 124, 0.34)`
- Text: `#f4fbf7`
- Muted text: `#9ab2a9`
- Emerald: `#12c878`
- Emerald glow: `rgba(18, 200, 120, 0.35)`
- Teal dark: `#06352b`

## Typography

- Use a clean geometric sans for all UI.
- Hero: 64-78px desktop, heavy weight, tight line height.
- Section headings: 48-56px desktop.
- Body: 16-18px, muted gray-green, relaxed line height.
- Buttons and labels: 14-15px, medium/semi-bold.

## Layout Patterns

- Max width around 1240px.
- Header floats inside the grid, logo left, simple nav center, pill CTA right.
- Hero is centered with trust pill, value proposition, CTA, proof points.
- Product console sits below hero as a large bordered panel.
- Feature cards use icon boxes, divider lines, and check bullets.
- Process section uses a vertical line with emerald nodes and side callouts.

## Interaction

- CTAs lift subtly on hover and receive emerald glow.
- Form/action panels use crisp disabled/loading states.
- Contract logs should look like operational telemetry.
- Avoid decorative blobs; the grid, lines, panels, and glow create the atmosphere.
